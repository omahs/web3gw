# Get Account Action

> Returns the account associated with the passed address. The address should be the string encoded version of a valid address.

- action name: !!btc.get.account
- parameters:
  - address [required]

## Example

```md
    !!btc.get.account
        address: b27a31f1b0af2946b7f582768f03239b1ec07c2c
```
